# Jogo da Forca - JavaScript

## Descrição

Este projeto consiste na implementação do jogo da forca utilizando HTML, CSS e JavaScript, sem utilização de banco de dados.

O objetivo do projeto foi desenvolver uma aplicação web simples, aplicando conceitos de programação, interação com o usuário, manipulação do DOM e versionamento de software utilizando Git e GitHub.

## Tecnologias Utilizadas

* HTML
* CSS
* JavaScript
* Git
* GitHub

## Funcionalidades

* Exibição da palavra oculta
* Verificação de letras corretas
* Contador de erros
* Sistema de vitória
* Sistema de derrota
* Seleção aleatória de palavras
* Reinício da partida
* Interface aprimorada
